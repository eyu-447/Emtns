# EMTNS
**A lightweight C tool to uncover hidden image metadata and binary details.**

## Features
- **Deep Extraction:** Gets image dimensions even if EXIF is stripped.
- **Full Metadata:** Camera info, GPS, and lens data (EXIF).
- **System Stats:** File size, permissions, and timestamps.
- **Pure C:** Blazing fast and minimal.

## Installation
Just run the installation script:

```bash
chmod +x install.sh
./install.sh
```

## Usage
```bash
emtns image.jpg
```
